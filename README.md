# Portfolio — Manas Agrawal

Personal portfolio website built with Next.js 16, TypeScript, and TailwindCSS.

**Live at [manasagrawal.online](https://manasagrawal.online)**

## What's inside

- 3D particle hero with interactive background
- Project showcase (Grahbook, FormLabs, CodeSnap)
- Services and process sections
- AI-powered terminal assistant
- Fully responsive, dark theme

## Tech stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** TailwindCSS
- **Animations:** Framer Motion
- **3D:** Three.js
- **Deployment:** Vercel

## Running locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Projects featured

- [Grahbook](https://grahbook.vercel.app) — WhatsApp commerce for Indian merchants
- [FormLabs](https://formlabs-eight.vercel.app) — Visual drag-and-drop form builder
- [CodeSnap](https://codesnap-eta.vercel.app) — AI-powered code screenshot tool
